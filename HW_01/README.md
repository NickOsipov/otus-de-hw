# Список навыков Data Engineer

Найти и изучить несколько ресурсов с вакансиями Data Engineer (hh.ru, работа.ру и др.). Выписать требования к навыкам и опыту работы с продуктами и технологиями. Обобщить.

## Технические навыки
- **Языки программирования:** Python, Scala
- **Базы данных:**
  - SQL: PostgreSQL, Greenplum, Clickhouse, MySQL
  - NoSQL: Cassandra, MongoDB, Redis
- **Инструменты обработки данных:** 
  - ETL/ELT: SQL, dbt, Apache Airflow, Dagster
  - Потоковая обработка: Kafka, Spark Streaming, Flink
  - Big Data: Hadoop

## Ключевые компетенции
- Разработка и оптимизация баз данных  
- Моделирование данных и проектирование архитектуры (Data Warehouse, Data Lake)  
- Анализ и обработка больших наборов данных, включая неструктурированные  
- Построение и организация ETL/ELT-процессов  
- Настройка процессов контроля качества данных (Data Quality)  
- Оптимизация SQL-запросов и повышение производительности  

## Профессиональные качества
- Аналитическое мышление и способность выявлять закономерности в данных
- Эффективная работа в команде
- Понимание бизнес-задач и их технической реализации
- Оперативное реагирование на запросы и изменения требований
